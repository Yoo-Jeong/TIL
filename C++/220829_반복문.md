## 220829 반복문

## `for`문 사용예시
```cpp
for (/*반복자 초기화*/; /*반복자 조건체크*/ ; /*반복자 변경*/)
{

}
```

```cpp
for (int i = 0; i < 3; ++i)
{
	printf("Output Test\n");
}

// 4번 반복(출력)
```


## `while`문 사용예시
```cpp
while(/*조건체크*/)
{

}
```

```cpp
int i = 0;
while (i < 3)
{
	printf("Output Test\n");
	++i;
}

// 3번 반복(출력)
```

## `continue` `break`

* `for`문과 `while`문 둘 다에 사용할 수 있다.  
* `continue`는 반복 구문이 수행되는 도중에 `continue`를 만나면 뒷부분은 더 이상 수행하지 않고 변경파트로 넘어가서 조건체크를 한다. 즉, **이번 반복횟수를 한 번 그냥 넘긴다.**
