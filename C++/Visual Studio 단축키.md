## Visual Studio 단축키 메모

## 단축키
|                      |                            |
|----------------------|----------------------------|
| 지정한 구문 주석       |  `Ctrl` + `k`, `c`        |
| 지정한 구문 주석 해제  | `Ctrl` + `k`, `u`          |
| `Alt Drag`           |                            |
| 함수 선언 보기        | `Ctrl` + `Shift` + `Space` |

<br/>

## 디버깅
|     |     |
| --- | --- |
| 디버깅 시작 <br/> 다음 중단점까지 코드 실행 | `F5` |
| 중단점 생성 및 해제 | `F9` |
| 디버깅 중, 구문 수행 | `F10` |
| 디버깅 중, 구문 수행(함수 진입) | `F11` |
| 디버깅 중, 디버깅 종료하기 | `Shift` + `F5` |
