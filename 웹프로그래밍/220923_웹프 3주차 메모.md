## 선언문
`<%!  %>` 느낌표가 있으면 선언문.

```Java
 <%@ page contentType="text/html;charset=EUC-KR"%>

 <h1>Declaration Example1</h1>

<%
	String name = team + " Fighting!!!";
%>

<%!
	String team = "Korea";
%>
출력되는 결과는 ? <%=name%>
```

<br>
