# 수학의 세계 공부

## 명제
p -> q

<br>

### 역
q -> p

<br>

### 이
~p -> q

<br>

### 대우
~q -> ~p

<br>

### 합동식
`a ≡ b (mod m)`   

(a를 m으로 나눈 나머지)와  
(b를 m으로 나눈 나머지)가 서로 같다.
