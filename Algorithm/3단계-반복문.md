* [2739 구구단](https://www.acmicpc.net/problem/2739)

`for문`과 입출력을 이용해서 구구단을 출력한다.  
`cin`과 `cout`를 사용하기 위한  
* `using namespace std;`
* `#include <iostream>` 표준입출력 함수를 사용하기 위한 헤더파일

```cpp
using namespace std;
#include <iostream>

int main()
{
	int N;
	cin >> N;

	for (int i = 1; i < 10; i++) {

		cout << N << " * " << i << " = " << N * i << "\n";

	}

}
```

* [8393 합](https://www.acmicpc.net/problem/8393)


```cpp
using namespace std;
#include <iostream>

int main()
{
	int n;
	cin >> n;
	int sum = 0;

	for(int i = 1; i <= n; i++)
	{
		sum = sum + i;
	}

	cout << sum;
}

```

using namespace std;

{

## [25304 영수증](https://www.acmicpc.net/problem/25304)

```cpp
using namespace std;
#include <iostream>

int main() {

	int X, N;
	int a, b;
	int sum = 0;

	cin >> X;
	cin >> N;

	for (int i = 1; i <= N; i++) {

		cin >> a >> b;	
		sum += (a * b);
	}


	if (sum == X) {

		cout << "Yes";
	}
	else {

		cout << "No";
	}

}
```

## [15552 빠른 A+B](https://www.acmicpc.net/problem/15552)

* `cin` `cout`을 사용하면 시간초과가 발생한다.
* 따라서 `scanf` `printf`를 사용한다.
* 왜 속도 차이가 나는지 알아보기!

```cpp
#include <stdio.h>;

int T;

int main(){
    scanf("%d", &T);
    
for(int i = 0; i < T; i++){
	int A, B;
	scanf("%d %d", &A, &B);
	printf("%d\n", A + B);
	}	
}
```
