# 10단계 - 정렬

#### 백준 단계별로 풀어보기

<br>

* [2750 수 정렬하기](https://www.acmicpc.net/problem/2750)


* 컴파일 에러

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int mian() {

	int num[1000];

	int N;
	cin >> N;

	for (int i = 0; i < N; i++) {
		cin >> num[i];
	}

	sort(num, num + N);

	for (int i = 0; i < N; i++) {
		cout << num[i] << "\n";
	}

	return 0;

}
```
