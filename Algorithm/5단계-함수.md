#### 백준 단계별로 풀어보기

<br>

* [15596 정수 N개의 합](https://www.acmicpc.net/problem/15596)

* long long자료형은 int보다 크다. 8바이트의 공간을 가지기 때문에.

```cpp
#include <vector>


long long sum(std::vector<int> &a) {

	long long ans = 0;
	for (int i = 0; i < a.size(); i++) {
		ans += a[i];
	}

	return ans;
}
```
